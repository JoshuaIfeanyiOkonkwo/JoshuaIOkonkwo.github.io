Autonomous Robot Vehicle Line Following & Obstacle Avoidance

Project Overview

This project is a compact autonomous robotic vehicle built using an Arduino, L298N motor driver, ultrasonic sensor, IR line sensors, and a servo‑mounted distance scanner. The robot can follow a line, detect obstacles, and make navigation decisions in real time.

1. Key Components

  • Arduino UNO: Main controller for all sensors and actuators
  • L298N Motor Driver: Controls two DC motors
  • HC‑SR04 Ultrasonic Sensor: Forward distance measurement
  • IR Sensors (x2): Line following detection
  • Servo Motor: Rotates ultrasonic sensor for side scanning
  • DC Motors + Wheels: Robot movement
  • 2×AA Battery Pack + Rocker Switch: Power supply and control

2. System Architecture

  • IR Sensors → Arduino → Motor Driver → Motors (line following)
  • Ultrasonic Sensor → Arduino → Servo Scan → Navigation Logic
  • Battery Pack → Rocker Switch → L298N → Arduino (Vin)

3. Core Logic

The robot combines line following and obstacle avoidance:

  • Both IR sensors detect line → move forward
  • Left or right IR triggered → turn accordingly
  • Obstacle detected < 10 cm → servo scans left/right → choose clearer path
